Free Code Camp Project: Image Search Abstraction Layer

This is a sample of an Abstraction Layer

Example usage:

regular search:

https://image-search-api-olbfoto.herokuapp.com/search/<keyword>

paginate search:

https://image-search-api-olbfoto.herokuapp.com/search/<keyword>?offset=<value>

Search History:

https://image-search-api-olbfoto.herokuapp.com/history/
